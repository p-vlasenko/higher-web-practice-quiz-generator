# Higher Web Practice — Quiz Generator

Генератор и проигрыватель квизов. Позволяет создавать квизы используя JSON-описание, сохранять их и проходить в браузере.

**Ключевые возможности**
- Генератор квизов из формата JSON (см. папку `quiz-examples/`).
- Просмотр списка созданных квизов.
- Проходить квизы в интерактивном режиме и просматривать результаты.

## Быстрый старт

Требования: Node.js (рекомендуется версия 24+).

Установите зависимости и запустите режим разработки:

```bash
npm install
npm run dev
```

Сборка для продакшена:

```bash
npm run build
npm run preview
```

## Структура репозитория (кратко)

- `src/` — исходный TypeScript код
	- `components/` — UI-компоненты и фабрики представлений
	- `presenters/` — логика презентации (MVP-подход)
	- `models/` — модели данных (квизовы, игры)
	- `pages/` — точки входа для страниц (`main.ts`, `quiz.ts`, `quizzes.ts`)
	- `utils/` — утилиты (storage, валидация, event-emitter и т.д.)
- `quiz-examples/` — примеры квизов в формате JSON (`html-quiz.json`, `css-quiz.json`, `js-quiz.json` и т.д.)
- `assets/`, `css/` — статические ресурсы и стили
- `index.html`, `quiz.html`, `quizzes.html` — шаблоны страниц

## Формат квизова (где смотреть примеры)

Для быстрого старта посмотрите примеры в `quiz-examples/`. Файлы содержат структуру вопросов, вариантов ответов и метаданные. Чтобы добавить новую квизову:

1. Создайте JSON-файл по образцу в `quiz-examples/`.
2. Подключите или загрузите файл через UI-генератор (или добавьте в сборку статически).

## техническая информация

- Код написан на TypeScript и собирается Vite.
- дизайн опирается на шаблон MVP (models, view, presenters).
- Основные команды:

```bash
npm install       # установить зависимости
npm run dev       # запуск dev-сервера (горячая перезагрузка)
npm run build     # сборка для продакшена
npm run preview   # локально просмотреть сборку
```
Файлы точек входа для страниц находятся в `src/pages/`.

## Где искать полезное

- Примеры квизов: `quiz-examples/`
- Точки входа: `src/pages/`
- Основная логика игры: `src/models/quiz-game.ts`
